Как получить куки Ozon? (30 секунд)
1. Откройте ozon.ru в браузере и войдите в аккаунт
2. Нажмите F12 → вкладка Application (или «Приложение»)
3. Слева: Storage → Cookies → https://www.ozon.ru
4. Найдите куку __Secure-access-token — скопируйте её значение
5. Или: вкладка Network → обновите страницу → любой запрос → вкладка Headers → найдите строку cookie: → скопируйте всё после неё
6. Вставьте в поле выше и нажмите «Сохранить»
